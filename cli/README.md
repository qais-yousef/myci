# Requirements

`pip3 install pyjen`

# Setup autocomplete for options

`pip3 install argcomplete`
`activate-global-python-argcomplete --user`
`complete -r myci`

You might need to add the following to your ~/.bash_completion to get it to
work:

```
for file in ~/.bash_completion.d/*
do
	. $file
done
```
